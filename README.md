# Minesweeper

This is a basics terminal based minesweeper game made using c++.

## How to Set it UP

Go to the [release page](https://github.com/Aryansjc/projects-on-c/releases/tag/exe) and click on the [minesweeper.exe](https://github.com/Aryansjc/projects-on-c/releases/download/exe/minesweeper.exe) and download it.

It will work only on windows this way for other os please run it manually as I have not found an easy way to express it online for all os.
